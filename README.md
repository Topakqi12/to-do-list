# Simple To‑Do List

A small, single‑page to‑do list web app built with **vanilla HTML, CSS, and JavaScript**. It lets you create tasks and sort them in different ways.

## Features

- **Create tasks** by typing in the input and pressing **Enter** or clicking **Add Task**.
- **Toggle completion** with the checkbox, or via the **Mark done / Mark active** button.
- **Edit a task** by **double‑clicking its title**.
- **Delete tasks** using the **Delete** button.
- **Sort tasks** using the toolbar:
  - Created: newest first
  - Created: oldest first
  - Title: A → Z
  - Title: Z → A
  - Status: incomplete first
- **Hide completed tasks** with the checkbox in the toolbar.

## Running the app

You can open it directly or serve it via a local web server.

### Option 1 – Open directly

1. Open the `index.html` file in your browser (double‑click it or drag it into a browser window).

### Option 2 – Run with a tiny dev server (Node required)

1. In a terminal, `cd` into this folder.
2. Run:

   ```bash
   npm install serve -g
   serve .
   ```

3. Open the printed local URL in your browser.
